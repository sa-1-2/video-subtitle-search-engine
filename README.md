# video-subtitle-search-engine

Steps to run the Video Subtitle Search Engine
``` python
conda create -p .searchengine python=3.10 -y
```
``` python
conda activate ./.searchengine
```
``` python
pip install -r requirements.txt
```
``` python
streamlit run app.py
```
